INSTALATION:

* Download nodejs and npm from the page https://www.npmjs.com/get-npm.

* Install nodejs in the computer. Make sure that "Add to PATH" feature is ON.

* Open the system command prompt (CMD) and run the command "npm install -g create-react-app"

* That's all you need to install the react packages. Now you can go to the project and start coding.

TO START THE PROJECT:

* Open the command promp and go to the project folder and run one of the command below:

	* Run the command "npm start" to start the development server.

	* Run the command "npm run build" to bundle the app into static files for production.

	* Run the command "npm test" to start the test runner.

	* Run the comand "npm run eject" to remove the tool and copy the build dependencies, configuration files and scripts into the app directory.
